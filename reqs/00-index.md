# Requerimientos - indice

Este directorio documenta los requerimientos del proyecto a partir de bible.md.

Convenciones
- ID: RQ-<AREA>-<NNN>
- Estado: planned | in-progress | done | blocked
- Version: v1 | v2 | backlog
- Dependencias: lista de IDs
- Medible: criterios de aceptacion verificables
- Checklist: tareas que componen el requerimiento

Areas
- VISION: vision, objetivo, alcance, principios
- ROUTE: rutas y render modes
- PDX: pokedex listado/detalle
- FILTER: filtros y query params
- DATA: data layer, caching, concurrencia, dedupe
- SSR: ssr/ssg/hydration/zoneless
- UI: ui/ux/tailwind/a11y/aria/cdk
- ERR: errores y seguridad
- TEST: testing
- PWA: service worker
- TOOL: tooling y ci
- LAB: labs
- DOC: docs
- STD: coding standards
- DELIV: entregables
- ADR: decisiones
- I18N: internacionalizacion
