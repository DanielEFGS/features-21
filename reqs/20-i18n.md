# Internacionalizacion

RQ-I18N-001
- Descripcion: Habilitar i18n nativo de Angular (v21) con locales definidos (p.ej. es/en) y proveedores de locale.
- Version: v1
- Dependencias: RQ-SSR-001
- Fundamento: Preparar la app educativa para múltiples idiomas sin romper SSR/hydration.
- Criterios de aceptacion:
  - LOCALE_ID configurable y locales registrados (date/number/currency).
  - SSR e hydration funcionan con el locale por defecto.
  - No se usan libs externas de runtime i18n en v1.
- Checklist:
  - [x] Registrar locale data y proveedor LOCALE_ID por defecto.
  - [ ] Verificar pipes/formatters funcionan en SSR y cliente.
  - [ ] Documentar idioma por defecto y cómo cambiarlo.

RQ-I18N-002
- Descripcion: Extraccion de mensajes y builds por locale con Angular localize.
- Version: v1
- Dependencias: RQ-I18N-001
- Fundamento: Generar bundles traducidos y mantener paridad SSR/CSR.
- Criterios de aceptacion:
  - `ng extract-i18n` genera `messages.xlf` y archivos por idioma.
  - `angular.json` tiene configuracion `localize` para cada locale (SSR y cliente).
  - Builds por locale producen rutas/outputs separados y funcionan en dev (`--localize`).
- Checklist:
  - [ ] Ejecutar extracción y crear archivos de traducción por idioma.
  - [ ] Configurar `angular.json` para builds locales (SSR/CSR).
  - [ ] Validar build y arranque en al menos dos locales.

RQ-I18N-003
- Descripcion: Cobertura de textos/labels/ARIA en vistas clave (Home, Pokedex, Labs, Docs).
- Version: v1
- Dependencias: RQ-I18N-001
- Fundamento: Asegurar accesibilidad y consistencia entre idiomas.
- Criterios de aceptacion:
  - Textos visibles y labels ARIA marcados con i18n; sin strings concatenados en templates.
  - Navegacion, modales (Docs) y filtros siguen funcionando con textos largos.
  - README explica cómo actualizar traducciones y verificar cambios.
- Checklist:
  - [ ] Marcar textos y ARIA en templates principales con `i18n`.
  - [ ] Probar UI con textos traducidos para desbordes/longitudes.
  - [ ] Actualizar README/Docs con pasos de traduccion.
